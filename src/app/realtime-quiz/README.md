# Realtime Quiz - 실시간 퀴즈쇼

Kahoot 스타일의 실시간 멀티플레이어 퀴즈 게임입니다.

## 📋 개요

- **타입**: 실시간 인터랙티브 퀴즈 게임
- **참여 방식**: 호스트(PC/프로젝터) + 참가자(모바일)
- **주요 기능**: 실시간 답변, 속도 보너스, 연속 정답 스트릭, 리더보드

## 🎯 핵심 기능

### 1. 퀴즈 템플릿
- **대한민국 지리 퀴즈**: 수도, 섬, 강 등
- **일반 상식 퀴즈**: 과학, 역사 등
- **영화 퀴즈**: 유명 영화와 감독
- **K-POP 퀴즈**: 한국 아이돌 그룹
- **과학 퀴즈**: 생물, 물리, 화학

### 2. 게임 플레이
1. 호스트가 퀴즈 선택 및 세션 생성
2. 참가자들이 코드로 참여
3. 카운트다운 후 게임 시작
4. 각 질문마다 타이머와 함께 진행
5. 정답 공개 및 점수 계산
6. 리더보드 업데이트
7. 최종 우승자 발표

### 3. 점수 시스템
- **기본 점수**: 100 ~ 1000점 (문제 난이도별)
- **속도 보너스**: 빠르게 답할수록 최대 50% 추가
- **연속 정답 보너스**: 연속 정답 시 10%씩 증가 (최대 50%)

## 🎨 UI/UX

### 호스트 화면 (PC/프로젝터)
- 질문 및 선택지 큰 화면 표시
- 4개 선택지를 색상+모양으로 구분 (빨강 삼각형, 파랑 다이아몬드, 노랑 원, 초록 사각형)
- 타이머 바와 남은 시간
- 참가자 답변 현황
- 리더보드 애니메이션

### 참가자 화면 (모바일)
- 2x2 그리드 답변 버튼 (한 손 조작 최적화)
- 색상+모양으로 빠른 선택 가능
- 실시간 타이머
- 정답/오답 즉시 피드백
- 획득 점수 표시

## 📁 파일 구조

```
src/app/realtime-quiz/
├── page.tsx                      # 메인 페이지 (메타데이터 + Suspense)
├── components/
│   ├── RealtimeQuizApp.tsx       # 메인 앱 컴포넌트
│   ├── HostView.tsx              # 호스트 프레젠테이션 화면
│   ├── ParticipantView.tsx       # 참가자 모바일 화면
│   └── Leaderboard.tsx           # 리더보드 (애니메이션)
├── store/
│   └── quizStore.ts              # Zustand 상태 관리
├── types/
│   └── index.ts                  # TypeScript 타입 정의
└── data/
    └── quizzes.ts                # 퀴즈 템플릿 데이터
```

## 🚀 사용 방법

### 호스트 (게임 진행자)
1. "호스트로 시작" 버튼 클릭
2. 퀴즈 템플릿 선택
3. 생성된 6자리 코드 공유
4. 참가자 대기
5. "게임 시작" 버튼으로 시작
6. 각 질문 후 "다음 문제" 진행

### 참가자
1. "참가자로 입장" 버튼 클릭
2. 닉네임 입력
3. 퀴즈 선택 (호스트와 동일)
4. 대기 화면에서 대기
5. 게임 시작 후 모바일로 답변
6. 리더보드에서 순위 확인

## 🎓 교육 활용 예시

### 수업 복습
- 교사가 수업 내용으로 퀴즈 진행
- 학생들이 모바일로 실시간 참여
- 이해도 즉시 확인 가능

### 팀빌딩
- 회사/팀 관련 퀴즈로 친목 도모
- 경쟁을 통한 참여도 증가
- 재미있는 학습 경험

### 이벤트
- 행사장 퀴즈쇼
- 경품 추첨 전 분위기 조성
- 참가자 인터랙션
