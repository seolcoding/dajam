# Changelog

All notable changes to the `@mini-apps/db` package will be documented in this file.

## [0.1.0] - 2024-12-22

### Added

- Initial release of `@mini-apps/db` package
- Supabase client initialization with environment variable support
- TypeScript type definitions structure (placeholder for generated types)
- Support for `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` environment variables
- Comprehensive README with usage examples
- EXAMPLE.md with React hooks, realtime, and authentication examples
- MIGRATION.md guide for integrating into existing apps
- Proper exports configuration for ESM
- Type-safe database access foundation

### Features

- Supabase JavaScript client (v2.47.10+)
- Environment variable validation with helpful warnings
- TypeScript support with declaration files
- Database type placeholder for future type generation
- Build configuration with tsup
- Source maps for debugging

### Documentation

- README: Installation and basic usage
- EXAMPLE: Advanced patterns (hooks, realtime, auth, Zustand)
- MIGRATION: Step-by-step integration guide
- CHANGELOG: Version tracking
